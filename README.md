# HomeStore

## Intro
I detta projekt ska vi skapa en e-handelssida med hjälp utav `React`och `TypeScript`. 

<br>

## Beskrivning
HomeStore är en e-handlessida där du kan handla hem olika möbler för just ditt drömhem.
